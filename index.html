Orbital edge computing adds computational intelligence to tiny, low-cost nanosatellites, enabling them to process remotely sensed data on orbit, without the need to communicate to Earth. Orbital edge computing puts machine learning and machine inference into space to process sensor data. This project's goal is to produce computational nanosatellite hardware and software designs, to build simulation and modeling tools for energy-constrained computational nanosatellite constellations, and to launch a computational nanosatellite constellation.

This project is the work of Ph.D. student Brad Denby at CMU ECE in the Abstract research lab led by Brandon Lucia.
